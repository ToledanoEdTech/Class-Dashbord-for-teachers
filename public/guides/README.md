| קובץ | תוכן |
|------|------|
| `guide-1.pdf` | שלב 1 – כניסה ליומן מחנך |
| `guide-2.pdf` | שלב 2 – הורדת קובץ ציונים |
| `guide-3.pdf` | שלב 3 – הורדת קובץ התנהגות |

המשתמשים יראו PDF אחד בכל פעם, עם כפתורי «הקודם» / «הבא».
