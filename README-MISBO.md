# מדריך אינטגרציה עם משוב

## סקירה כללית

המערכת תומכת כעת בהתחברות אוטומטית למשוב והורדת קבצים. המערכת משתמשת ב-web scraping עם Puppeteer כדי לבצע את הפעולות אוטומטית.

## דרישות

1. **שרת Backend** - צריך להריץ את השרת מתיקיית `server/`
2. **Node.js** - גרסה 18 ומעלה
3. **גישה למשוב** - שם משתמש וסיסמה תקינים

## התקנה והפעלה

### 1. התקנת תלויות השרת

```bash
cd server
npm install
```

### 2. הפעלת השרת

```bash
npm run dev
```

השרת ירוץ על `http://localhost:3001`

### 3. הגדרת ה-Frontend

וודא שה-variable `VITE_API_URL` מוגדר ב-`.env`:

```env
VITE_API_URL=http://localhost:3001
```

או שהקוד ישתמש ב-default `http://localhost:3001`.

### 4. הפעלת ה-Frontend

```bash
npm run dev
```

## שימוש

1. פתח את האתר
2. בחר "התחברות למשוב" במקום "העלאה ידנית"
3. הזן:
   - כתובת משוב (URL)
   - שם משתמש
   - סיסמה
4. לחץ על "התחבר והורד קבצים"
5. המערכת תנסה להוריד את הקבצים אוטומטית

## מגבלות

- המערכת תלויה במבנה של משוב - אם המבנה משתנה, ייתכן שצריך לעדכן את הסלקטורים
- לא כל אתרי משוב זהים - ייתכן שצריך להתאים את הקוד למימוש הספציפי
- הורדות אוטומטיות לא תמיד עובדות - במקרה כזה, השתמש במצב העלאה ידנית
